Om valgfrie parametere og plot()-funksjonen

En ting som er kjekt kunne fr vi gr videre er valgfrie parametre. Dette blir ofte brukt i store funksjoner som har potensiale til gjre mye forskjellig. Da nsker man gjerne ha mye fleksibilitet, men nsker kanskje ikke mtte skrive ned hva man nsker p alle de ulike innstillingene. Da bruker man parameter som har en default verdi.

For lage en slik funksjon skriver man ned hva default verdien er i funksjons deklarasjon. Deretter nr man skal bruke funksjonen kan man velge hvor mange paramter man nsker endre p. Du trenger ikke full forstelse av dette, men m vite at det gr an. Se p kodecellen under for et eksempel fr du gr videre.

In[1]:
def optional_function(arg0 = 5, arg1 = 10, arg2 = 15):
    print(f"{arg0 = } | {arg1 = } | {arg2 = }")

optional_function()
optional_function(1, 2, 3)
optional_function(1, arg2 = 5)
optional_function(arg1 = 7)

arg0 = 5 | arg1 = 10 | arg2 = 15
arg0 = 1 | arg1 = 2 | arg2 = 3
arg0 = 1 | arg1 = 10 | arg2 = 5
arg0 = 5 | arg1 = 7 | arg2 = 15

Mange funksjoner vi bruker i Python har slike optional-parametre, og plot()-funksjonen til matplotlib er en slik funksjon. For gi kurven vi skal plotte et spesielt utseende, kan vil legge til en formateringsstreng med en rekke nkkelsymbol som et tredje argument i plot()-funksjonen. For eksempel s vil flgende kommando plotte kurven gitt ved punktene i x og y som en prikkete linje med bl farge:

plt.plot(x, y, ':b') # Tegn kurven som en prikkete bl linje

Vi skal n se p ulike "ekstra-instrukser" vi kan gi denne funksjonen for pynte p grafen vr.

Farge

Farge er relativt intuitivt forholde seg til, ihvertfall nr man holder seg til basisfargene gitt i tabellen under. Da kan man angi en farge ved inkludere n enkelt bokstav i formateringsstrengen.

Bokstav Farge
'b' Bl
'g' Grnn

|'r' | Rd |'c' | Lysebl (Cyan)| |'m' | Lilla (Magenta)| |'y' | Gul| |'k' | Svart | |'w'| Hvit |

Du kan lese mer om fargene som er tilgjengelig her.

a)

Gitt funksjonen $f(x) = \sin(x)$, lag et plot som viser kurven til $f(x)$ i intervallet $0\leq x \leq 10$ der fargen p kurven er svart.

In[2]:
### BEGIN SOLUTION
import numpy as np
import matplotlib.pyplot as plt

def f(x):
    return np.sin(x)

x = np.linspace(0, 10, 101)
plt.plot(x, f(x), 'k')
#plt.plot(x, g(x))
plt.savefig("Figurer/oppg7a_LF.png")
fig = plt.gcf()
### END SOLUTION

Linjestil

Man kan ogs endre linjestilen til grafene i matplotlib.

Nr man plotter bruker man argumentet ls for linjestil, og color for farge

Nr man bruker ls kan man velge mellom 4 alternativer

Symboler Linjestil
'-' n kontinuerlig linje
':' Prikkete linje
'--' Strekete linje
'-.' Streker og prikker

b)

Gitt funksjonen $g(x) = \cos(x)$, lag et plot som viser kurven til $f(x)$ i intervallet $0\leq x \leq 10$ der linjestilen er en prikkete linje.

In[4]:
### BEGIN SOLUTION
import numpy as np
import matplotlib.pyplot as plt
def g(x):
    return np.cos(x)
x = np.linspace(0, 10, 101)

plt.plot(x, g(x), ':')
plt.savefig("Figurer/oppg7b_LF.png")
fig = plt.gcf()
### END SOLUTION

Punktmarkrer

Av og til kan det vre nyttig markere punktene som man tenger en kurve gjennom. Da kan man legge til et symbol i formateringsstrengen som oppgir hvordan de ulike punktene skal markeres. Det er veldig mange mulige det gr an velge mellom, men her er et utvalg:

Symbol Punktmarkr
'o' Sirkel $\circ$
'v' Trekant $\triangledown$
'd' Diamant $\diamond$
'+' Kryss $+$

Merkelapper

Med flere kurver i samme figur kan det vre nyttig med en merkelapp for differensiere mellom kurvene. Dette gjres i to steg:

  1. Angi teksten i en merkelapp ved bruke optional parameter label
  2. La merkelappene vises i plottet ved kalle funksjonen plt.legend()
    • PS: Det er ogs mulig spesifisere hvorf merkelappene skal vises ved skrive posisjonen som et optional argument til funksjonen plt.legend(). Eksempel: plt.legend(loc="upper center")

Eksempel:

In[8]:
x = np.linspace(-3, 3)
y1 = x**2
y2 = 0.2*x**3
plt.plot(y1, label="f(x) = x^2")
plt.plot(y2, label="f(x) = 0.2*x^3")
plt.legend()

d)

Lag et plot som viser kurven til $f(x)$ fra deloppgave a) og $g(x)$ fra deloppgave b) i tillegg til punktene i deloppgave c). Kurvene skal merkes med forklarende merkelapper.
Til info: punktene fra deloppgave c skal markere krysningspunktene til funksjonene $f(x)$ og $g(x)$.

In[9]:
### BEGIN SOLUTION
import numpy as np
import matplotlib.pyplot as plt

def f(x):
    return np.sin(x)
    
def g(x):
    return np.cos(x)

x = np.linspace(0, 10, 101)
plt.plot(x, f(x), 'k', label="f(x)")
plt.plot(x, g(x), ':', label="g(x)")

x = np.array([1, 5, 9])*np.pi/4
y = np.sqrt(0.5)*np.array([1, -1, 1])
plt.plot(x, y, 'd', label="Krysningspunkt")
plt.legend()
plt.savefig("Figurer/oppg7d_LF.png")
fig = plt.gcf()
### END SOLUTION
